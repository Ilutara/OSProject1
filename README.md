Current files and their roles:
logger.py -- responsible for looging commands
encrypt.py -- responsible for encryption, decryption, and password operations
main.py -- driver program responsible for running the program

Instructions to run the program:
1. Make sure all files are in the same folder
2. Run the command "python3 main.py fileName", where fileName can be replaced with the intended name of the log file
